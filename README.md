# Classic ML: Car Price Prediction 🚗

This project focuses on predicting car prices using classical machine learning techniques. It serves as a baseline for comparing traditional regression models with more complex neural networks.

## 🛠 Tech Stack
- **Python 3.11**
- **Scikit-learn** (Linear Regression, Random Forest)
- **Pandas & NumPy** (Data processing)
- **Matplotlib & Seaborn** (EDA - Exploratory Data Analysis)

## 📈 Roadmap
- [x] Initial project structure

- [ ] Advanced models (XGBoost/Random Forest)

## 🚀 How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run training script: `python src/train.py`
